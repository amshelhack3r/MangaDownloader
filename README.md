## Installation

1. Clone: `git clone https://github.com/amshelhack3r/MangaDownloader.git'
2. CD into it: `cd MangaDownloader`
3. Create a virtual environment(skip if you already have)
```shell
$ pip install pipenv
$ pipenv shell
$ pipenv install
```
3. Run program
```shell 
$ python main.py
```
4. Enjoy :D
